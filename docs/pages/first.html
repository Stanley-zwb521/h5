第四课¶

如果你完成了第三课的练习，请继续下面的内容。 本篇内容包括：

课前预习：

  1. 预习本课件第4章节

课堂讲解：

在本课程中，你将随着讲师的实战操练逐渐的掌握JavaScript的知识。讲师会现场演示JavaScript的主题内容，你将深刻理解这些主题的实际应用场景。课堂介绍的主题包括以下内容。

  1. Destructuring（重构）

  2. Array helper methods（操作数组的方法）

  3. Variables hoisting

  4. Let, var, and const

  5. Rest and spread operators

  6. The ternary operator（三元运算符）

  7. Default parameters（默认参数）

  8. Closures（闭包）

  9. IIFEs (Immediately Invoked Function Expressions)

  10. Classes （类和对象）

课后练习：

  1. 完成练习：https://github.ibm.com/fsd-training/h5-course/tree/master/assignment/lession4

  2. 更新学员练习进度表：https://pages.github.ibm.com/fsd-training/h5-course/

本篇内容包括：

课前预习：

  1. 预习本教材全部内容知识。

SBA（Skill based assessment）练习：

  1. 通过本练习，巩固本次培训的全部教学知识：

  2. 掌握H5、CSS、JavaScript的基本技能；

  3. 学会GitHub的操作，并且学会在GigHub上搭建自己的网站；

  4. 学会使用BootStrap设计网页；

  5. 学会使用JavaScript，完成CRUD的操作；

  6. 搭建个人RestFul API 服务，学习如何与HTTP交互；

  7. 学习如何测试及部署自己的应用程序。

  1. 完成练习：https://github.ibm.com/fsd-training/h5-course/tree/master/assignment/sba-assignment

  2. 更新学员练习进度表：https://pages.github.ibm.com/fsd-training/h5-course/

  3. 作业提交截至日期：2020年9月9日23:00

  4. 作业效果图

答辩：

  1. 时间：计划2020年9月11日，星期五，下午3:00，举行答辩活动。

  2. 参与人员：完成作业学员及其PEM，PM，CVITube邀请的评委及往期的学员；

H5答辩活动坚持公开、公平及公正的原则，现场评委打分，评选的标准公示如下：

  1. 个人答辩。个人打开自己的GitHub URL（注意不是本地的URL），1-2分钟的讲解，从前到后的演示，功能越多，分值越高（几个功能，就对应几分，因此个人需要准备好环境）；

  2. 控制台不能有错误；（1个错误减5分）；

  3. 页面UI美观（元素对齐、布局错位、输入框验证、错误提示、字体效果，共5项，满分10分，每缺一项扣2分）；

  4. 统计结论正确：统计出下期概率最大的一组数字；（过程清晰10分，结论正确10分）；

  5. 评委提问前6次作业的内容（不超过2个问题，满分5分）。

  6. 评委打分表

最后，评出前3名，给予奖励。（人均：300 Blue point）
